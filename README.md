# Jones Home Test

## Task 1 - Playwright Automation

Automates filling a contact form at https://test.netlify.app/ and verifies reaching the thank you page.

### Requirements

- Node.js
- Playwright + Zod (`npm install`)
- Chromium (`npx playwright install chromium`)

### Run

```bash
node index.js
```
